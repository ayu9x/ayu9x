## 📐 DevOps Architecture Flow

```
Developer Push
      │
      ▼
┌─────────────────┐     ┌─────────────────┐     ┌──────────────────┐
│  GitHub Actions  │────▶│  Docker Build   │────▶│  Image Registry  │
│  CI Pipeline     │     │  & Test Suite   │     │  ECR / DockerHub │
└─────────────────┘     └─────────────────┘     └────────┬─────────┘
                                                          │
                         ┌────────────────────────────────▼─────────┐
                         │         Kubernetes Cluster (EKS/GKE/AKS) │
                         │  ┌──────────┐  ┌──────────┐  ┌────────┐  │
                         │  │  Pod 1   │  │  Pod 2   │  │ Pod 3  │  │
                         │  └──────────┘  └──────────┘  └────────┘  │
                         └────────────────┬──────────────────────────┘
                                          │
              ┌───────────────────────────┼──────────────────────────┐
              ▼                           ▼                           ▼
     ┌────────────────┐        ┌────────────────┐       ┌────────────────────┐
     │   Prometheus   │        │    Grafana      │       │  CloudWatch/Datadog │
     │   + Alerting   │        │   Dashboards    │       │  Centralized Logs  │
     └────────────────┘        └────────────────┘       └────────────────────┘
```
